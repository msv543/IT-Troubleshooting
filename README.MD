# IT Troubleshooting Labs

A collection of hands-on IT troubleshooting labs completed in my home lab using Windows Server 2025, Windows 11, Active Directory, and VMware Workstation Pro.

These labs simulate real-world support scenarios by intentionally introducing common IT issues, identifying the root cause, documenting the investigation process, and verifying the solution.

Each lab follows a consistent troubleshooting methodology:

- Problem
- Symptoms
- Investigation
- Commands Used
- Root Cause
- Resolution
- Verification

## Purpose

This repository documents practical troubleshooting experience by recreating common IT support incidents in a controlled lab environment. Each exercise focuses on identifying the underlying cause, applying the appropriate resolution, and validating the fix using industry-standard troubleshooting techniques.
